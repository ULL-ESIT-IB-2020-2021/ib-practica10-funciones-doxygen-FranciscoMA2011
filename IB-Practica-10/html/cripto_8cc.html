Detailed Description

El programa le permite encriptar y desencriptar ficheros mediante el método xor o César. El método César consiste en cambiar cada caracter por uno desplazado un cierto número de posiciones ASCII El método xor consiste en aplicar la operación xor a cada uno de los caracteres, con una clave dada.
